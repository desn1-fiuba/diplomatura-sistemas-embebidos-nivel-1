# Diplomatura en Sistemas Embebidos - Nivel 1

Repositorio público de materiales, consignas, ejemplos de código y recursos de apoyo para la Diplomatura en Sistemas Embebidos - Nivel 1.

## Materias

La diplomatura está organizada en tres materias:

1. Introducción a los Sistemas Embebidos.
2. Periféricos y Comunicaciones en Sistemas Embebidos.
3. Taller de Trabajo Final.

## Plataforma de trabajo

- Placa de desarrollo: NUCLEO-F446RE.
- Entorno de desarrollo: STM32CubeIDE.
- Lenguaje de programación: C.
- Herramientas auxiliares: Git, GitHub, terminal serie y osciloscopio.

## Organización del repositorio

Cada materia cuenta con una carpeta propia donde se organizarán las clases, diapositivas, consignas, ejemplos y recursos correspondientes.
