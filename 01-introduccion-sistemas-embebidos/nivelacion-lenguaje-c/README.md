# Nivelación en lenguaje C

Esta sección contiene ejercicios de práctica en lenguaje C destinados a estudiantes que necesitan nivelar conocimientos previos de programación antes de avanzar con el desarrollo de firmware sobre microcontroladores.

El material también puede ser utilizado por estudiantes que ya conocen el lenguaje, pero desean repasar o reforzar conceptos fundamentales.

## Objetivos

- Repasar tipos de datos, operadores y expresiones.
- Practicar estructuras de control condicionales e iterativas.
- Desarrollar funciones simples y reutilizables.
- Trabajar con arreglos, cadenas y estructuras.
